# Vendor Dispute Email Style

Use a concise, professional tone.

Required content:

- Invoice number.
- PO number when available.
- Specific discrepancy.
- Request for a corrected invoice or written confirmation.

Template:

Subject: Invoice {invoice_number} requires correction

Hi {vendor_name},

We reviewed invoice {invoice_number} and cannot approve it for payment yet.

Issues found: {reasons}.

Please send a corrected invoice or confirm the discrepancy so we can continue processing.

Best,
AP Operations
